# csv-uploader-nodejs
This simple nodejs application takes data from uploaded csv, and present it as a sortable dashboard on a webpage.

### If you using docker and docker-compose:
```
$ docker-compose build
$ docker-compose up
```
### Else you need nodejs installed locally and run the following
```
$ npm install
```
### For development environment
```
$ npm run start:dev
```
### For production
```
$ npm start
```
### To run tests
```
$ npm test
```
